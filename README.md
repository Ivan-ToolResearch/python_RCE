遠端執行程式碼攻擊（RCE）
=================
遠端執行程式碼攻擊又稱為Remote Code Execution（簡稱RCE），也就是利用服務中可以讓使用者輸入（input）的區塊，進行指令的運行，如此一來，等於能夠控制server的整臺電腦。本專案將用簡單的範例掩飾駭客的攻擊思維，以及如何較有效的進行防禦。
